## Cognitive Bandwidth Router Prototype

The `cognitive_router` package under `src/` is a runnable scaffold for routing AI-agent tasks to humans based on SLO risk, model confidence, and an attention/load signal derived from lightweight telemetry. See `docs/cognitive_bandwidth_router.md` for the architecture overview, simulation instructions, and next steps.
